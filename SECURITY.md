# Security Policy

## Supported Versions

This repository contains documentation and business-related information.
Security updates apply to the latest version available in the `main` branch.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ Yes    |
| Older  | ❌ No     |

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

### How to Report
- Open a **GitHub Issue** with the label `security`, **OR**
- Contact the repository owner directly

### What to Expect
- A response within **48–72 hours**
- Verification of the reported issue
- Fix or mitigation if required

Please do **not** publicly disclose vulnerabilities before they are reviewed.

Thank you for helping keep this project secure.
